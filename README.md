
## Responsive Breakpoints

- **Mobile**: 320px - 767px
- **Tablet**: 768px - 1024px
- **Desktop**: 1025px - 1440px
- **Large Desktop**: 1441px+

## Mobile Optimizations

1. **Touch-Friendly Controls**: Larger buttons (minimum 44x44px)
2. **Performance**: Reduced particle counts on mobile
3. **Orientation Support**: Landscape and portrait modes
4. **Safe Area Support**: Notch and home indicator areas
5. **Reduced Motion**: Respects user's motion preferences

## Browser Support

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+
- Mobile browsers (iOS Safari, Chrome for Android)

## Performance Features

- Lazy loading where applicable
- Reduced animations on low-power devices
- Efficient canvas rendering
- CSS containment for better rendering performance

## Running the Website

Simply open `index.html` in any modern web browser. No build process or dependencies required.
